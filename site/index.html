What’s inside

Down to the tab

Open by design

If it’s open on your Mac, you can teach Jay to reach it.

A plugin is a manifest and an executable that speaks JSON. Any language. Ship support for your favorite app in an afternoon.

# plugin.json: the whole contract
{
  "apiVersion": 1,
  "name": "Spotify",
  "targetApp": "Spotify",
  "exec": "adapter"
}

# ./adapter list  →  JSON on stdout
[ { "id":"37i9", "title":"Deep Focus", "active":true } ]

# ./adapter activate 37i9  →  switch to it

Twenty lines, any language.

  • Two commands: list and activate. Shell, Python, Swift, Go. Your call.
  • Runs out-of-process in parallel, with a hard timeout. A slow plugin only drops itself.
  • On when you add them, off with one toggle. Plugins you install work immediately; disable any in Preferences whenever you like.
  • Ships with Terminal & VS Code adapters as copy-paste starting points.
Generate one with AI 
Build a plugin for Jay (a macOS launcher). A plugin is a folder with two files:

1) plugin.json
{ "apiVersion": 1, "name": "<App Name>", "targetApp": "<exact app name in the menu bar, or null for always-on>", "exec": "adapter", "supportsClose": false }

2) An executable "adapter" (any language; chmod +x) that answers these commands:
 • adapter list          → print a JSON array to stdout:
   [ { "id":"stable-unique-id", "title":"required", "subtitle":"optional", "url":"https://… (optional → favicon)", "group":"optional section", "active": false } ]
 • adapter activate <id>   → focus that item in the app (AppleScript / CLI / URL scheme); print nothing.
 • adapter close <id>      → optional; close/dismiss that item. Set "supportsClose": true only if you implement it.

Rules: keep `list` fast (< 1s) and read-only; print [] when the app isn't ready.

Full spec + a copy-paste example: https://github.com/spacegrowth/jay/tree/main/plugins  (read plugins/README.md; start from plugins/terminal).

My app: <NAME>. How to read its open items and focus one: <AppleScript / local API / URL scheme>.

Yours, and only yours

The stuff that sees all your windows should keep its mouth shut.

On-device intelligence

The AI that groups your Contexts is Apple’s on-device model. No network call, no cloud, no Private Cloud Compute. It degrades to a deterministic grouper on older macOS.

No phone-home

No accounts, no analytics, no crash pings, no server, anywhere in the app. There’s nothing to opt out of because there’s nothing being sent.

One honest exception

The only thing that ever leaves your Mac is a favicon lookup, a bare domain, exactly like any browser. And you can switch even that off.
